# **App Name**: Zuckky AI

## Core Features:

- AI-Powered Video Editing: Use Gemini API and Submagic API through Make.com to understand user instructions and apply edits to videos, leveraging AI to determine when and how to apply video edits to achieve the desired outcome using the provided video and Submagic's tool. 
- Welcome Chat Response: Provide a welcoming message and instructions for new users on how to start editing their videos when they start a chat with text.
- Guided Editing Workflow: Implement a conversation flow that guides users through uploading, template selection, custom training (if applicable), additional instructions, and confirmation.
- Instruction Summary & Confirmation: Summarize user requests and video editing plans before execution, prompting the user to confirm before proceeding.
- Video Upload & Preview: Enable users to upload video files and preview the small-size thumbnail after upload, also restrict upload to only video file.
- Video Editing Processing Display: Display a collapsible 10-step sequential list with a progress bar during video editing to indicate that the editing process is happening.
- Canvas Preview & Download: Automatically open a closable/reopenable 'Preview Canvas' from the right with the ability to preview and download the edited video.

## Style Guidelines:

- Primary color: Calm neon green (#39FF14) for a modern and tech-forward feel.
- Background color: Dark gray (#222222) to create a premium and modern aesthetic.
- Accent color: Soft gray (#CCCCCC) for UI elements and highlights, ensuring legibility.
- Body and headline font: 'Inter', a sans-serif font for a clean and contemporary look.
- Use consistent and professional icons for all buttons and functions.
- Implement a clean and responsive design with a collapsible sidebar.
- Use subtle and gentle animations for transitions and loading states to provide a smooth user experience.
